# restaurant_management
Veri Yapıları dersi kapsamında üç kişilik ekip tarafından geliştirilmiş Restoran Sipariş Yönetimi Otomasyonu. Proje Java dili kullanılarak yazılmıştır ve restoranlarda siparişlerin eklenmesi, takibi ve yönetimini sağlayacak şekilde tasarlanmıştır. Kullanıcı dostu arayüzü ile siparişlerin hızlı ve kolay yönetimi hedeflenmiştir.
